# Query Prompt
1. Create a branch in github using mcp-tool
2. Genereate code for a login page and show
3. Do commit the changes in the branch with proper commit_message, files_data and branch_name. Fix it from step2.
4. Raise PR this branch to `main` using mcp-tool
5. Get PR file using mcp-tool
5. Do PR review and add reveiw comments using mcp-tool
6. Fix the review comments and commit to the branch using mcp-tool
7. Send back for PR review until all the critical comments are fixed using mcp-tool
